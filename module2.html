Module 2: Data Structures & Algorithms

Covers complexities, standard data structures, searching/sorting, and puzzle-like challenges found in interviews.

Lesson 1: Linked Lists & Iterators

Implement singly/doubly linked lists. Understand iterator design, especially in custom containers. Tasks:

  • Create a LinkedList class with push_back, pop_front, etc.
  • Implement a forward iterator for your list.

Lesson 2: Sorting & Searching

Dive into QuickSort, MergeSort, and searching algorithms (binary search). Tasks:

  • Code a quicksort with random pivot selection.
  • Implement a binary search on a sorted array.

Lesson 3: Graphs & BFS/DFS

Represent graphs with adjacency lists, perform breadth-first and depth-first search. Tasks:

  • Implement BFS that returns shortest distance from a start node.
  • Use DFS to detect cycles in a directed graph.

Lesson 4: Common Algorithmic Puzzles

Typical problems like two-sum, sliding window maxima, etc. Tasks:

  • Given an array and a target, return indices of two numbers that sum to target.
  • Sliding window maximum for each window of size k in an array.
