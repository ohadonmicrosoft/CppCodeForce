Module 2: Data Structures & Algorithms

This module explores fundamental and advanced data structures—arrays, linked lists, stacks, queues, trees, graphs, and the core sorting/searching algorithms.

Lesson 1: Arrays & Linked Lists

Understand the implementation and trade-offs of arrays vs. linked lists. Implement basic operations (insert, delete, traversal) for each, and discuss performance.

Lesson 2: Trees & Graphs

Explore tree traversals (in-order, pre-order, post-order) and implement BFS/DFS for graphs. Apply these to solve basic problems in data structures.
