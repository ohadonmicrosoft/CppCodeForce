Module 3: Real-Time & Embedded

Delve into RTOS basics, embedded concurrency, interrupt handling, and hardware-level debugging strategies.

Lesson 1: RTOS & Task Scheduling

Understand priority-based scheduling, rate-monotonic vs. earliest deadline first, and how tasks switch in RTOS.

  • Write pseudocode for a simple round-robin scheduler vs. a priority-based scheduler.
  • Discuss how interrupts interact with task scheduling in real-time constraints.

Lesson 2: ISRs & Concurrency

In embedded contexts, concurrency often revolves around interrupts. Learn how to safely share data between main tasks and ISRs without corrupting state.

  • Implement an ISR that updates a global counter; use volatile to avoid compiler optimizations.
  • Discuss disabling interrupts briefly vs. using lock-free structures.

Lesson 3: Hardware Interaction & Debugging

Explore memory-mapped I/O, controlling peripherals (GPIO, SPI, I2C), and debugging with tools like JTAG or logic analyzers.

  • Simulate reading/writing from a memory-mapped peripheral address using a volatile pointer.
  • Describe how you'd trace a communication error with a logic analyzer.
