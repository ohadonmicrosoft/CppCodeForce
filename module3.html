Module 3: Real-Time & Embedded

Explores RTOS concepts, concurrency in embedded contexts, interrupts, hardware-level debugging, etc.

Lesson 1: RTOS & Task Scheduling

Understand how real-time schedulers prioritize tasks, manage deadlines, and handle context switching. Tasks:

  • Write pseudocode for a priority-based scheduler.
  • Explain rate-monotonic scheduling vs. earliest-deadline-first.

Lesson 2: ISRs & Concurrency in Embedded

Handling interrupts safely, shared data with volatile, and concurrency pitfalls in microcontrollers. Tasks:

  • Write an ISR that toggles an LED and increments a counter.
  • Demonstrate lock-free data exchange between main loop and ISR.

Lesson 3: Hardware Interaction & Debugging

Using GPIO, SPI, or I2C in embedded code. Approaches to low-level debugging with JTAG or logic analyzers. Tasks:

  • Simulate reading/writing a hardware register with a volatile pointer.
  • Describe how you’d debug a bus fault with a logic analyzer trace.
