Module 6: Templates & Metaprogramming

Learn advanced template usage, partial specialization, SFINAE, and compile-time tricks that power many modern libraries.

Lesson 1: Function Templates & Class Templates

Basic function/class templates let you write generic code with type parameters.

  • Create a max function template that returns the bigger of two values.
  • Implement a GenericArray class template with operator[] for indexing.

Lesson 2: Template Specialization & Partial Specialization

Certain templates need specialized handling for specific types. This is common in trait libraries.

  • Create a PrettyPrint class template, then specialize for std::string to handle quotes.
  • Show partial specialization for pointer types, e.g., PrettyPrint.

Lesson 3: SFINAE & Type Traits

Substitution Failure Is Not An Error (SFINAE) underpins many advanced template techniques. std::enable_if and type traits let you conditionally compile code based on type properties.

  • Implement a function that only compiles if T is integral using std::enable_if.
  • Use std::is_same, std::is_integral, or std::is_pointer in your code logic.
