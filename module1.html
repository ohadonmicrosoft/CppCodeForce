Module 1: Foundations & Modern C++

This module covers essential modern C++ techniques—including RAII, smart pointers, and lambdas—to build a solid foundation.

Lesson 1: RAII & Constructors

Learn how resource acquisition is tied to initialization in C++.

  • Create a FileWrapper class that opens a file in the constructor and closes it in the destructor.
  • Handle errors gracefully.

Lesson 2: Smart Pointers & Move Semantics

Discover how to use std::unique_ptr and std::shared_ptr along with move semantics for efficient memory management.

  • Refactor raw pointer code to use smart pointers.
  • Demonstrate transferring ownership with std::move.

Lesson 3: Lambdas & STL Algorithms

Use lambdas in combination with STL algorithms (e.g., std::remove_if) for concise code.

  • Implement a lambda with std::remove_if.
  • Compare capture methods.
