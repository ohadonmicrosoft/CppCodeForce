Module 1: Foundations & Modern C++

This module covers essential modern C++ techniques—including RAII, smart pointers, and lambdas.

Step 2: Smart Pointers & Move Semantics

Figure: RAII Diagram

Lesson 1: RAII & Constructors

Learn how resource acquisition is tied to initialization in modern C++. Implement a FileWrapper class that opens a file in its constructor and closes it in the destructor, handling errors gracefully.

Lesson 2: Smart Pointers & Move Semantics

Discover how to use std::unique_ptr and std::shared_ptr along with move semantics to manage resources effectively. Transfer ownership with std::move and avoid memory leaks.

Lesson 3: Lambdas & STL Algorithms

Use lambdas in combination with STL algorithms (e.g., std::remove_if) for concise yet powerful code. Explore different capture methods and their real-world scenarios.
