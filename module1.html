Lesson 1: RAII & Constructors

Concept: Resource Acquisition Is Initialization (RAII) ensures resource ownership is bound to object lifetimes.

  • Create a FileWrapper class that opens a file in the constructor and closes it in the destructor.
  • Consider exception-handling in the constructor—what if the file can’t be opened?

Lesson 2: Smart Pointers & Move Semantics

Use std::unique_ptr, std::shared_ptr, std::weak_ptr for dynamic allocations. Move semantics eliminates expensive copies.

  • Rewrite a raw pointer approach to use std::unique_ptr.
  • Demonstrate using std::move to transfer ownership cheaply.

Lesson 3: Lambdas & STL Algorithms

Lambdas combined with std::transform, std::remove_if, etc. can make code more concise and expressive.

  • Use std::remove_if with a lambda to remove all negative numbers from a vector.
  • Explore captures by value vs. by reference in a lambda.
