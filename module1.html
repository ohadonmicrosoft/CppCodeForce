Module 1: Foundations & Modern C++

This module covers essential modern C++ techniques: RAII, smart pointers, lambdas, and more. Master these features to write safer, more expressive code.

Lesson 1: RAII & Constructors

Concept: Resource Acquisition Is Initialization (RAII) ensures resource ownership is bound to object lifetimes. If an object goes out of scope, its destructor frees the resource. This pattern greatly reduces memory leaks and fosters exception safety.

  • Create a FileWrapper class that opens a file in the constructor and closes it in the destructor.
  • Consider exception-handling in the constructor—what if the file can’t be opened?

Lesson 2: Smart Pointers & Move Semantics

Modern C++ introduces std::unique_ptr, std::shared_ptr, and std::weak_ptr for managing dynamic allocations. Move semantics eliminates expensive copies, letting you “move” data efficiently.

  • Rewrite a raw pointer approach to use std::unique_ptr (ownership pattern).
  • Demonstrate using std::move to transfer ownership of an object to another scope.
  • Explore a scenario where std::shared_ptr is useful (reference counting) vs. std::unique_ptr (exclusive ownership).

Lesson 3: Lambdas & STL Algorithms

Lambdas enable concise inline function objects. Pair them with powerful STL algorithms (std::transform, std::remove_if, etc.) to write higher-level, cleaner code.

  • Use std::remove_if with a lambda to remove all negative numbers from a std::vector.
  • Demonstrate capturing external variables by value vs. by reference in a lambda.
  • Use std::transform to apply a function to each element in a list, doubling the values.
