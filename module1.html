Module 1: Foundations & Modern C++

This module covers essential modern C++ techniques: RAII, smart pointers, lambdas, and more. Mastering these paves the way for advanced concurrency, memory management, and embedded topics.

Lesson 1: RAII & Constructors

Concept: Resource Acquisition Is Initialization (RAII) ensures resources (files, network sockets, dynamic memory) are owned by objects, automatically released in destructors. This greatly reduces leaks and makes code more exception-safe.

  • Implement a class FileWrapper that opens a file in the constructor and closes in the destructor.
  • Handle potential exceptions in the constructor properly (e.g., if the file fails to open).

Lesson 2: Smart Pointers & Move Semantics

Modern C++ offers std::unique_ptr, std::shared_ptr, and std::weak_ptr to manage dynamic allocations. Move semantics lets you transfer ownership cheaply, avoiding deep copies.

  • Convert raw pointer usage to std::unique_ptr in a sample code snippet.
  • Show how to pass ownership with std::move, and discuss safe usage patterns.
  • Experiment with std::shared_ptr for reference counting. Consider how std::weak_ptr prevents reference cycles.

Lesson 3: Lambdas & STL Algorithms

Lambda expressions let you define inline function objects. Combine them with powerful STL algorithms (std::for_each, std::transform, std::remove_if, etc.) to write concise, high-level code.

  • Use std::remove_if with a lambda to remove all even numbers from a vector.
  • Explore captures by value vs. by reference in lambdas. How does each affect the code?
  • Use std::transform to apply a function to all elements in a container.
