Module 1: Foundations & Modern C++ Features

A refresher on essential C++ syntax, RAII principles, and modern features introduced in C++11/14/17. Mastering these sets the stage for more advanced topics later.

Lesson 1: RAII & Constructors

RAII ensures resources are acquired in constructors and released in destructors, preventing leaks. Tasks:

  • Create a class that opens and closes a file within constructor/destructor.
  • Handle exception safety: ensure no leak occurs if an exception is thrown.

Lesson 2: Smart Pointers & Move Semantics

Learn std::unique_ptr, std::shared_ptr, std::weak_ptr, and how move semantics can eliminate unnecessary copies. Tasks:

  • Port a raw pointer approach to unique_ptr.
  • Show an example transferring ownership with std::move.

Lesson 3: Lambda Expressions & STL Algorithms

C++ lambdas let you write inline functions. The STL offers std::sort, std::for_each, std::transform, etc. Tasks:

  • Use a lambda to filter out even numbers in a vector.
  • Demonstrate capturing variables by reference vs. by value.
