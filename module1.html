Lesson 1: RAII & Constructors

Concept: Resource Acquisition Is Initialization ensures resources are owned by objects.

  • Create a FileWrapper class that opens a file in constructor & closes in destructor.
  • Handle exceptions if file fails to open.

Lesson 2: Smart Pointers & Move Semantics

Use std::unique_ptr, std::shared_ptr, std::weak_ptr, & move semantics for efficient memory management.

  • Rewrite raw pointer approach with unique_ptr.
  • Show std::move transferring ownership cheaply.

Lesson 3: Lambdas & STL Algorithms

Combine lambdas with std::remove_if, std::transform, etc. to write concise code.

  • Use std::remove_if with a lambda to remove negative numbers.
  • Explore captures by value vs. by reference.
