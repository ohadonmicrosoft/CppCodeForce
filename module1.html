Module 1: Foundations & Modern C++

This module covers essential modern C++ techniques—such as RAII, smart pointers, and lambdas—to build a solid foundation.

Lesson 1: RAII & Constructors

Concept: Resource Acquisition Is Initialization ensures that resources are acquired and released safely.

  • Create a FileWrapper class that opens a file in its constructor and closes it in the destructor.
  • Implement proper error handling when opening files.

Lesson 2: Smart Pointers & Move Semantics

Learn how to use std::unique_ptr, std::shared_ptr, and move semantics for effective memory management.

  • Refactor raw pointer usage with unique_ptr.
  • Demonstrate the use of std::move to transfer ownership.

Lesson 3: Lambdas & STL Algorithms

Use lambdas with STL algorithms (like std::remove_if and std::transform) for concise code.

  • Implement a lambda function with std::remove_if.
  • Compare capture-by-value versus capture-by-reference.
