Module 4: Concurrency & Multithreading

Threads, mutexes, condition variables, lock-free structures—essential knowledge for advanced systems-level C++.

Lesson 1: std::thread & std::mutex

Launch multiple threads in C++. Use std::mutex or std::lock_guard to prevent data races.

  • Write a simple program that spawns threads to increment a shared counter.
  • Show how missing locks cause data corruption (race conditions).

Lesson 2: Condition Variables & Futures

Explore std::condition_variable for thread synchronization. Understand std::future and std::promise for asynchronous tasks.

  • Implement a producer-consumer queue using a std::condition_variable to block the consumer if empty.
  • Show how std::async can launch tasks that return std::future.

Lesson 3: Lock-Free Data Structures

Lock-free programming aims to avoid mutexes for high performance. Typically uses std::atomic operations to ensure correctness.

  • Discuss compare_exchange_strong usage in implementing a lock-free stack.
  • Explain ABA problems and possible solutions (std::atomic, hazard pointers, etc.).
